# Engineering Computation Notebooks

**EIT (Engineer-in-Training) Foundational Math & Physics**

> A collection of Python-based Jupyter notebooks demonstrating mastery of foundational engineering principles (Statics, Thermodynamics, Circuits) required for the FE exam, solved using modern computational tools (`numpy`, `sympy`).

---

## 🎯 Core Technical Competencies

| Capability                           | Deliverable                                                                   |
| :----------------------------------- | :---------------------------------------------------------------------------- |
| Engineering Fundamentals (Thermodynamics, Circuits, Statics) | Python Jupyter notebooks solving standard FE exam problems. |
| Mechanical & Physical Engineering | Notebooks demonstrating static FEA analysis or kinematic math. |

---

## 📚 Notebooks Included

*   **`notebooks/statics_and_thermodynamics_fe_prep.ipynb`**: Solves a 2D truss analysis using the Method of Joints (Statics) and calculates the final temperature of an isentropic gas expansion (Thermodynamics).
